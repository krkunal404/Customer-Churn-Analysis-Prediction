# 📘 Customer Churn Analysis & Prediction

## 📊 About the Project

This project explores a telecom dataset to understand why customers leave (churn) and how we can predict it. The idea is to use data analysis and simple machine learning to find patterns and make useful predictions.

---

## 🎯 What I Did

* Cleaned and prepared the dataset
* Explored the data using graphs and statistics
* Identified key factors affecting churn
* Grouped customers based on risk
* Built a model to predict churn

---

## 📈 Key Insights

* Customers with **low tenure** are more likely to leave
* **Higher monthly charges** increase churn chances
* **Month-to-month contracts** have the highest churn
* Long-term customers are more stable

---

## 📊 Analysis Performed

* Data cleaning and handling missing values
* Multiple visualizations (bar charts, histograms, etc.)
* Correlation analysis
* Hypothesis testing (t-test)
* Customer segmentation

---

## 🤖 Prediction Model

I used a **Logistic Regression model** to predict whether a customer will churn or not.
The model gives a decent accuracy and shows that churn can be predicted using basic features.

---

## 📌 Conclusion

This project shows that pricing, contract type, and customer duration play a big role in churn. Companies can reduce churn by focusing on new customers and improving pricing strategies.
